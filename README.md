# Windows 11 Clean Installation Guide

A complete, illustrated technical user guide for performing a clean installation of Windows 11 — covering system requirements, bootable USB creation, the full step-by-step setup process, post-installation configuration, troubleshooting, and FAQs.

## Contents

- `Windows11_Clean_Install_Guide.docx` — the full guide (Word format)

## What's Inside

1. **Introduction** — what a clean install is and who this guide is for
2. **System Requirements** — CPU, RAM, storage, TPM 2.0, Secure Boot, and firmware prerequisites
3. **Preparation** — backing up data, gathering drivers, and creating bootable USB installation media
4. **Step-by-Step Installation** — booting from USB, language setup, product key entry, license terms, drive partitioning, automated install, and out-of-box experience (OOBE)
5. **Post-Installation Configuration** — updates, drivers, activation, and restoring data
6. **Troubleshooting** — common errors like "Windows cannot be installed on this drive," USB boot issues, and TPM/Secure Boot detection problems
7. **FAQs** — 11 common questions on product keys, accounts, data loss, timing, and more

Each major step includes an illustrative diagram to make the process easy to follow.

## Usage

Download `Windows11_Clean_Install_Guide.docx` and open it in Microsoft Word or any compatible word processor (LibreOffice Writer, Google Docs, etc.).

> **Tip:** After opening in Word, right-click the Table of Contents and select **Update Field** to populate page numbers.
